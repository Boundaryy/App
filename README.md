## 💫 커밋 컨벤션
| type | description |
| :---: | :---: |
| DOCS | 새로운 기능, 컴포넌트 등 새로운 것이 추가되었을 때 |
| FIX | 기존의 코드에서 잘못된 부분을 수정할 때 |
| REFACTOR | 기존의 있는 코드를 리팩토링 하였을 때 |
| DELETE | 기존의 있던 내용을 삭제하였을 때 |
| STYLE | 스타일을 변경하는 상황일 때 |
